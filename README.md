# kserve-practice

KServe practice workspace organized by purpose.

## Structure

- `docs/`: setup, metrics, and profiling notes
- `scripts/`: runnable helper scripts
- `config/`: input and runtime configuration files
- `results/`: generated experiment or profiling results
- `k8s/manifests/`: Kubernetes and KServe manifests
- `third_party/`: downloaded external distributions and vendor assets
- `.cache/`: local generated cache files
